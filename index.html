Rendering Mode Selects which low-llevel rendering mode to use.
DirectX 11 is the older graphics API.
Performance is a mode that offers significant performance gains by disabling most rendering features to reduce memory usage and lighten the load on CPU and GPU.
DirectX 12 is a new graphics API and may offer performance increases over DirectX 11. It is required to enable Nanite (which dynamically and efficiently scales meshes in the envioronment) and Lumen (which uses multiple ray-tracing methods to solve global illumination and reflections). Some options from DirectX 11 do not appear in performance mode, but apply anyways.
